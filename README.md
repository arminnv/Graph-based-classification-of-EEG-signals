# Graph-Based Classification of EEG Signals
This repository contains code for my senior project "Graph-Based Classification of EEG Signals", where I implemented and tested various connectivity-based methods for EEG classifaction, as well as developing my own new methods. The algorithms are graph feature extraction and classification, common spatial patterns (FKT), graph learning based on smoothness, FKT on Graph-Fourier Transform, Graph-Convolutional Networks, and FKT on GFT of class-specific graphs. I was able too out perform all the previous methods on small BCI dataset aquired from spinal cord injury and stroke patients achieving an accuracy of 89.5% for two-class classification. I am currently writing a paper on this project and detailed material and reports will be available soon.
